# Download Historical Data

Historical data can be downloaded using the following steps:

1. Change MT4 settings to increase the number of data that can be saved
2. Download historical data

Please also refer to the following external links.

[MT4 Convenient Function ① How to Download Historical Data](https://www.gaitamefinest.com/mt4_convenient_function_list_post1)

[How to Get FXDD Historical Data and Procedures](https://gemgemfx.com/fxdd-historical-data/)

> [!NOTE]
> Please note that only MetaQuotes data can be downloaded with this procedure.

## 1. Change MT4 Settings to Increase the Number of Data That Can Be Saved

1. Click "Tools" → "Options" in MT4.
2. In the "Options" window, select "Charts", fill "Max bars in history" and "Max bars in chart" with "9999..." (fill with "9"s), and click "OK".

![AltText {priority}{768x432}](/img/option_history_bar_change.png)


Next, download the data from the History Center.

## 2. Download Historical Data


1. Click "History Center" from "Tools" in MT4.

2. The "History Center Window" will open, so click "Group 1" or "Group 2" to expand.
A list of currency pairs will be displayed within the expanded "Group 1" or "Group 2", so double-click the required currency pair to display a list of timeframes.
With a timeframe selected, click "Download".

3. Next, you will be asked "The downloaded historical data differs from the data stored on the trade server. Therefore, this data will not match the past prices of 'Gaitame Finest Company Limited'. If you want to continue anyway, please press OK." Click "OK".

4. When you click, the download will start and a green bar will extend to the right.
(Please wait until the download is complete.)
When the bar reaches the right end, the download is complete.

![AltText {priority}{768x432}](/img/history_download.png)

When you restart MT4 (close MT4 with the "×" in the upper right once, then open MT4 again), the downloaded data will be in a state where it can be displayed on the chart.
