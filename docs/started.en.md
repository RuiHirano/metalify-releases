# Metalify Getting Started Guide

Before you begin implementing Metalify, please follow the steps below to set up. This guide introduces the basic implementation steps. For detailed procedures and troubleshooting, please refer to the detailed pages linked below.

## 1. Subscribe to a Plan

First, you need to subscribe to a Metalify plan.

1. Log in to the Metalify website.
2. Select "Subscription" from the side menu.
3. On the displayed page, purchase your desired plan.

For details about plans and subscription methods, please refer to [**Subscribe to a Plan**](/docs/plan).

![AltText {priority}{768x432}](/img/plan_no_subscription.en.png)

## 2. Install the Application

Next, download and install the Metalify application.

1. Download the app from the Metalify website.
2. Run the downloaded file and proceed with the installation.
3. Once installation is complete, launch the app.
4. In the launched app, enter your email address and password to log in.

For detailed installation methods and how to use the app, please refer to [**Install the App**](/docs/install-app).

![AltText {priority}{768x432}](/img/download.en.png)

## 3. Configure MetaTrader4

Configure the settings to integrate Metalify with MetaTrader4.

1. Launch MetaTrader4.
2. Select Options from the Tools menu.
3. Click the "Expert Advisors" tab and enable the option to allow DLL usage.

For detailed MetaTrader4 settings and how to integrate with Metalify, please refer to [**Configure MT4 Settings**](/docs/mt4-settings).

![AltText {priority}{768x432}](/img/allow_dll.png)

This completes the basic implementation of Metalify. Refer to the detailed guides in each section to learn about further configuration and how to utilize the application.
